# Remote Housing Crisis Web Application

## Overview
The Remote Housing Crisis Web Application is a web application developed using the Django framework. It was created for the HIT237 Assessment 2. It is intended to address the remote housing crisis by providing a system that can manage and display housing availability in remote and regional areas.
